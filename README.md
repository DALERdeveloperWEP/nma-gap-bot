# nma-gap-bot

## Loyihani sozlash

1. virtual muhut yarartish: `python -m venv <name>`
2. virtual muhutni ishga tushurish: 
    - windows uchun: `source <name>/Scripts/activate`
    - mac/linux uchun: `source <name>/bin/activate`
    - venv ni o'chirish: `deactivate`
3. kerakli kutubxonalarni o'rnatish: `pip install <package>`
4. requirements.txt filega packeges larni yozish: `pip freeze > requirements.txt`
5. env (environment variable) file yaratish: `touch .env`
